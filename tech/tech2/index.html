    អាសន្ន! NSA ប្រាប់ឲ្យអ្នកប្រើ Windows 7/XP ប្រញាប់អាប់ដេតឡើង Windows ថ្មីជាបន្ទាន់

    Posted by Veasna WT

    យោ

    ​ង​តាម​សារព័ត៌មាន BGR បាន​ចុះ​ផ្សាយ​ភ្នាក់ងារ​សន្តិសុខ​ជាតិ​អាមេរិក NSA ​បាន​ចេញ​សារ​ព្រមាន​និង​ដាស់តឿន​ឲ្យ​អ្នក​ប្រើប្រាស់​ប្រព័ន្ធ​ប្រតិបត្តិ Windows ជំនាន់​ចាស់ ត្រូវ​ប្រញាប់​ដំឡើង​ទៅ​កាន់ Windows 10 ​ជា​បន្ទាន់ ដើម្បី​បញ្ចៀស​បញ្ហា​ដ៏​ធំ​មួយ។

    នោះ​គឺ​មេរោគ​វាយប្រហារ​ថ្មី​ឲ្យ​ឈ្មោះ​ថា Bluekeep ​ដែល​វា​ត្រូវ​បាន​គេ​ប្រៀបធៀប​ទៅ​នឹង WannaCry ដែល​ជា​ការវាយប្រហារ​ដ៏​ធំ​មួយ​នៅ​ឆ្នាំ ២០១៧ បង្ក​ការ​ខាត់បង​ទ្រង់ទ្រាយ​ធំ​ដល់​ទៅ​ខ្ទង់​ពាន់​លាន​ដុល្លារ​ទូទាំង​សកលលោក។

    Bluekeep គឺ​ជា​ការ​វាយប្រហារ​ម្យ៉ាង​ដែល​អាច​បើក​ផ្លូវ​ឲ្យ Hacker អាច​ជ្រៀត​ចូល​នូវ​មេរោគ​តាម​រយៈ​ការ​បញ្ជា Remote Desktop Services ដោយ​តាម​របាយការណ៍​ពី Microsoft មាន​ម៉ាស៊ីន​ជិត ១ លាន​គ្រឿង​ដែល​កំពុង​រង​ហានិភ័យ​ខ្ពស់ ជា​ពិសេស​គឺ​ម៉ាស៊ីន​ដំណើរការ​ដោយ​ប្រព័ន្ធ​ Windows 7 និង XP។

    ក្នុង​សេចក្តី​ប្រកាស​ផ្លូវការ​របស់ NSA បាន​បញ្ជាក់​ថា៖ យើង​ណែនាំ​ឲ្យ​មនុស្ស​រាល់​គ្នា​ឆ្លៀត​ពេល​និង​ធនធាន​ក្នុង​ការ​ Update ទៅ​កាន់ OS ជំនាន់​ក្រោយ​បង្អស់ ដោយ​នេះ​ពិត​ជា​សំខាន់​មិន​ត្រឹម​តែ​ការពារ​ប្រព័ន្ធ​សន្តិសុខ​ជាតិ តែ​សម្រាប់​ប្រព័ន្ធ​ទាំង​អស់​ក្នុង​លោក។
