    ច្បាស់ការ! Huawei Mate X ប្រើ Android តម្លៃ ២ ៦០០ ដុល្លារនឹងមានដាក់លក់នៅខែកញ្ញា

    Posted by Veasna WT

    កា

    ល​ពី​ខែ​កុម្ភៈ​កន្លង​ទៅ​ក្រុមហ៊ុន Huawei ​បាន​ប្រកាស​ផ្សាយ​នូវ​ស្មាតហ្វូន​បត់​ Mate X ​ដែល​ដំបូង​ឡើយ​មាន​គម្រោង​ចង់​ចេញ​លក់​នៅ​ចុង​ខែ​មិថុនា ប៉ុន្តែ​ដោយ​សារ​រកាំរកូស​បញ្ហា​សង្រ្គាម​ពាណិជ្ជកម្ម​ជាមួយ​អាមេរិក​ក៏​លើក​ពន្យារពេល​ទៅ​ពេល​ក្រោយ។

    ​​​​​​យ៉ាងណាមិញ​យោង​តាម​ដំណឹង​ច្បាស់ការ​ពី CNBC និង The Wall Street Journal ​ដែល​ទទួល​បាន​ពី​អ្នក​នាំ​ពាក្យ​របស់​ក្រុមហ៊ុន Huawei ​បញ្ជាក់​ប្រាប់​ថា Mate X នឹង​ដាក់​លក់​លើ​ទីផ្សារ​សកល​នា​ខែ​កញ្ញា​ខាង​មុខ ហើយ​ចុច​គោលដៅ​ទៅ​លើ​ប្រទេស​ដែល​បាន​រៀបចំ​រចនាសម្ព័ន្ធ​ 5G ជា​ទីផ្សារ​ចម្បង។

    ប្រភព​បាន​សង្កត់​ធ្ងន់​ថា​ម៉ូឌែល​នេះ​នឹង​ពុំ​រង​ឥទ្ធិពល​ពី​ការ​ហាមប្រាម​របស់​រដ្ឋាភិបាល​អាមេរិក​នោះ​ឡើយ ពោល​គឺ​នៅ​តែ​ដំណើរការ​ប្រព័ន្ធ Android និង​កម្មវិធី​របស់ Google ជា​ធម្មតា ​ព្រោះ Mate X ​បង្ហាញ​ខ្លួន​មុន​កាលបរិច្ឆេទ​ដែល​ក្រុមហ៊ុន​ជាប់​ឈ្មោះ​ក្នុង​បញ្ជី​ខ្មៅ។ ទាក់ទិន​នឹង​តម្លៃ Mate X ​ថ្លៃ ២ ២៩៩ អឺរ៉ូ​ឬ​ប្រមាណ ២ ៦០០ ដុល្លារ​អាមេរិក៕
